== Description ==
PCM Ads plugin is designed to display google ads, leaderboard and to integrate webskin on your site.

== Installation ==
1. Upload "pcm-ads" plugin to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Update your settings on "PCM Ads" setting page.
